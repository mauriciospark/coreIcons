# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2024

### Adicionado
- Interface web completa com busca em tempo real
- Suporte a 300+ ícones de tecnologias e linguagens
- Modal de detalhes com cópia de URL/nome
- Lazy loading de imagens
- Fallback para ícones não encontrados
- Responsividade para mobile

### Ícones Incluídos
- Linguagens: JavaScript, TypeScript, Python, Java, C, C++, C#, Go, Rust, Ruby, PHP, Swift, Kotlin e mais
- Frameworks: React, Angular, Vue, Svelte, Next.js, Nuxt.js, Django, Laravel
- Ferramentas: Docker, Kubernetes, Git, VS Code, Vim
- Cloud: AWS, Azure, GCP, Vercel, Netlify
- E muitos outros...

### Tecnologias
- HTML5 semântico
- CSS3 com Flexbox e Grid
- JavaScript vanilla (ES6+)
- Sem dependências externas
